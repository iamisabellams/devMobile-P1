# Aventura 1: RPG React Native - Clone de Telas

## Objetivo da Aventura:

-   Reproduzir duas telas reais de um aplicativo conhecido.
-   Utilizar arquivos Styles.
-   Utilizar componentes básicos do React Native.
-   Entregar o projeto organizado em um repositório Git.
-   Incluir imagens do app escolhido para a clonagem.

Então, nessa primeira aventura, recriamos duas telas:

### Tela de Login do aplicativo Skoob

O Skoob é uma rede social brasileira para leitores. Nele, você pode registrar os livros que já leu, está lendo ou quer ler, acompanhar metas literárias, fazer resenhas, participar de clubes de leitura e interagir com outros leitores. Funciona como uma estante digital onde você organiza suas leituras e descobre novos livros.

### Tela de perfil do aplicativo Spotify

O Spotify é uma plataforma de streaming de música e podcasts. Você pode ouvir milhões de faixas, criar playlists personalizadas, acompanhar artistas favoritos e receber recomendações baseadas no seu gosto. É usado tanto para música quanto para conteúdos falados, como podcasts, entrevistas e audiolivros.
